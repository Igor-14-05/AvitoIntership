# AvitoIntership

# Тест-кейсы для платформы модерации объявлений

## 1. Десктопная версия

### 1.1 Главная страница (/ и /list) — Фильтр «Диапазон цен»

AVITO-FILTR-1.1.1
Название: Проверка работы фильтра диапазона цен (минимальная и максимальная цена)  
Предусловия: Открыта главная страница, есть объявления с разными ценами.  
Шаги:
1. Ввести значение в поле «Цена от» (например, 1000).
2. Ввести значение в поле «Цена до» (например, 5000).
3. Нажать «Применить» / дождаться автообновления списка.
4. Проверить, что в результатах все объявления имеют цену в указанном диапазоне (вкл. граничные значения).
5. Очистить фильтр и проверить, что список вернулся к исходному состоянию.

Ожидаемый результат: Фильтр корректно ограничивает список. При очистке — все объявления отображаются снова.  
Постусловия: Фильтр сброшен.

AVITO-FILTR-1.1.2  
Название: Негативные сценарии диапазона цен (от > до, отрицательные значения, буквы)  
Шаги: Аналогично выше + попытка ввести некорректные данные (10000 в «от», 5000 в «до»; -100; «abc»).  
Ожидаемый результат: Валидация (ошибка, блокировка кнопки, игнорирование) или корректное сообщение.

### 1.2 Главная страница — Сортировка «По цене»

AVITO-SORT-1.2.1
Название: Проверка сортировки объявлений по цене (возрастание/убывание)  
Шаги:
1. Выбрать сортировку «По цене (возрастание)».
2. Проверить порядок цен в списке (неубывающий).
3. Переключить на «По цене (убывание)».
4. Проверить порядок (невозрастающий).

Ожидаемый результат: Объявления отсортированы корректно. При смене направления порядок меняется.

### 1.3 Главная страница — Фильтр «Категория»

AVITO-FILTR-1.3.1   
Название: Проверка мульти/одиночного выбора категорий  
Шаги:
1. Выбрать одну или несколько категорий.
2. Применить фильтр.
3. Убедиться, что в результатах только объявления из выбранных категорий.
4. Сбросить фильтр.

Ожидаемый результат: Фильтрация работает точно по выбранным категориям.

### 1.4 Главная страница — Тоггл «Только срочные»

AVITO-TOGGL-1.4.1 
Название: Проверка тогла «Только срочные»  
Шаги:
1. Включить тоггл.
2. Проверить, что в списке только срочные объявления (наличие специальной метки или признака).
3. Выключить тоггл — должны появиться обычные объявления.

Ожидаемый результат: Тоггл корректно фильтрует список.

### 1.5 Страница статистики (/stats) — Управление таймером обновления

AVITO-TIMER-1.5.1   
Название: Кнопка «Обновить» (ручное обновление)  
Шаги:
1. Открыть /stats.
2. Нажать «Обновить».
3. Проверить, что данные статистики обновились (время последнего обновления изменилось, цифры изменились или хотя бы таймстамп).

AVITO-TIMER-1.5.2 
Название: Кнопка остановки таймера  
Шаги:
1. Запустить автообновление (если не запущено).
2. Нажать «Остановить таймер».
3. Подождать предполагаемое время обновления.
4. Убедиться, что данные не обновляются автоматически.

AVITO-TIMER-1.5.3 
Название: Кнопка запуска таймера  
Шаги:
1. Остановить таймер.
2. Нажать «Запустить таймер».
3. Подождать интервал — данные должны обновиться автоматически.

Ожидаемый результат: Все три элемента управления работают стабильно, без ошибок в UI/консоли.

## 2. Мобильная версия

AVITO-MOBILE-2.1
Название: Переключение между тёмной и светлой темой  
Шаги:
1. Открыть сайт на мобильном разрешении (или в devtools — device toolbar).
2. Найти переключатель темы.
3. Переключить на тёмную тему → проверить цвета фона, текста, карточек.
4. Переключить обратно на светлую.
5. Проверить, что переключение сохраняется (при перезагрузке страницы, если есть).

Ожидаемый результат: Тема меняется корректно, контраст достаточный, нет сломанных элементов.
